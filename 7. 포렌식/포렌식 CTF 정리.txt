1. 툴

 분석
 - Audacity: 소리 이미지화 분석
 - Autopsy: 무료 포렌식 분석 툴
 - encase: 미국 수사기관 포렌식 툴
 - exiftool: 메타데이터 분석
 - Forensic explorer: 유료 포렌식 분석 툴
 - forevid: 영상 포렌식, 프레임 단위 저장 가능
 - kakaodecrypt: 카카오톡 DB 분석
 - kape: 아티팩트 추출
 - LECmd: 링크 파일 분석
 - live response collect: 휘발성 데이터 추출
 - Network Miner: 네트워크 패킷 분석
 - NTFS log tracker: NTFS 로그 분석
 - RBCmd: 휴지통 분석
 - REGA: GUI 웹 히스토리 분석
 - RegistryExplorer: 레지스트리 분석
 - SSview: 문서 구조 출력 (은닉 파일 탐지)
 - Sysinternals suite: 윈도우 모든 기능 분석
 - Videocleaner: 정보 없음
 - Volatility: 메모리 분석
 - Winprefetchview: 프리패치 분석
 - WinsearchDBanalyzer: 파일 구조, 인터넷 방문 페이지, 접속 기록 등 분석
